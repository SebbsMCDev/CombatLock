# CombatLock

**CombatLock** is a fully configurable combat tagging plugin for Minecraft Paper 1.21.x.

## Features
- Combat timer with action bar countdown
- Blocks commands during combat
- Logout punishment kills player
- Fully configurable via `config.yml` and `messages.yml`
- Clean, modular, professional code

## Installation
1. Download the JAR
2. Place it in `plugins/`
3. Start the server to generate configs
4. Edit `config.yml` and `messages.yml` as needed
